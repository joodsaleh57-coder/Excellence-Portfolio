# Excellence Portfolio App - PRD Summary

## Concept
A mobile app (adapted from a Chrome extension concept) to capture evidence, manage improvement plans, and generate structured PDF reports for educational platforms like Madrasati and Noor.

## Target Users
- School Leader
- Excellence Coordinator
- Vice Principal
- Teacher
- Student Counselor

## Core User Journey
1. **Capture Evidence**: Select indicator -> Choose category -> Save (URL, file, screenshot, note).
2. **Close Indicator**: Validate checklist -> Submit for review -> Approve.
3. **Export PDF**: Select scope -> Preview -> Generate -> Download.

## UI Structure (Mobile Adaptation)
- **Dashboard**: Visit readiness, overdue tasks, indicators without evidence.
- **Evidence Capture**: Interface to add URLs, files, screenshots, or notes.
- **Indicators**: View titles, domains, current vs target levels, evidence count, and plan status.
- **Tasks**: My tasks (today/week), filters, quick completion.
- **Evidence Center**: Centralized view of all captured evidence.
- **Export**: Generate PDF reports (Full school file, domain report, single indicator, weekly report).
- **Settings**: App configurations.

## MVP Features
- Dashboard with key metrics.
- Smart Evidence Capture.
- Templates: Workshop report, meeting minutes, classroom visit, results analysis, remedial plan, initiative.
- Improvement Plans: Actions, owner, deadline, status.
- Governance: Pre-approval checklist, Audit log, Protection rules (approved evidence is view-only).

## PDF Structure
- Cover page, index, domains, indicators, evidence grouped by category, improvement summary.
